# MALARIA-OMICS AFRICA

Plateforme génomique clinique pour la surveillance du paludisme en Afrique de l’Ouest.

## MVP (v0.1)
- Pipeline génomique *Plasmodium falciparum* (résistance : pfcrt, pfmdr1, kelch13)
- Rapport clinique PDF (1–2 pages)
- API Django + dashboard web léger

## Données
Prototype utilisant uniquement des données publiques (SRA/ENA) pour la démonstration.
