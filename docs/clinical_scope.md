# Clinical scope (MVP)

This MVP focuses on *Plasmodium falciparum* genomic analysis for surveillance of antimalarial drug resistance markers.

## Included
- FASTQ -> QC -> alignment -> variant calling -> targeted annotation
- Target genes: pfcrt, pfmdr1, kelch13
- Generation of a short clinical-style PDF report (informational)

## Not included (MVP)
- Clinical decision support / prescription
- Patient-identifiable data storage
- Transcriptomics / proteomics integration

## Disclaimer
The generated reports are informational and must not replace medical judgment.
